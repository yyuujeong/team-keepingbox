## :mag_right: 주요 기능
- 담당 부분: 메인 페이지 및 예약하기 페이지 UI/UX 설계
- 오늘 날짜 이전은 선택되지 않도록 구현
- 사이즈, 날짜에 따라 요금 자동 계산
- 자바스크립트로 입력값을 검증하고, 경고 및 확인 창을 통해 잘못된 입력에 의한 오류를 방지
- 공통부분을 모듈화하고자 ES6 모듈로 변경하여 유지 보수성과 안전성 개선
